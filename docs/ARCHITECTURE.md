# Architecture

## Overview

Balance is a static web project made of HTML pages, CSS stylesheets, and JavaScript modules.

## High-Level Structure

- Entry pages (for example `index.html`, `balance.html`, `takeoff.html`) provide UI screens/tools.
- JavaScript files implement calculator, data, UI, and domain logic.
- CSS files provide styling for page layouts and controls.
- `images/` and `resources/` store static assets.

Update this file when module boundaries or page responsibilities materially change.
